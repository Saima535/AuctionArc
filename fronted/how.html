How It Works

1. Register

Create account as Buyer or Seller

2. Browse / List

Buyers browse auctions, sellers create listings

3. Place Bid

Manual bid or auto-bid system

4. Win & Pay

Winner pays securely via escrow

5. Delivery

Seller ships item, buyer confirms receipt
